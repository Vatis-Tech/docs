---
title: "Get API access"
description: "Get the API key to gain access to the Vatis Streams API"
---

<Steps>
    <Step title="Sign up">
        Create an account on the Vatis Tech cloud platform. You can sign up [here](https://vatis.tech/app).
    </Step>
    <Step title="Get the API key">
        Once you have signed up, you can get the API key from the [API Dashboard](https://vatis.tech/app/playground/transcribe).
    </Step>
    <Step title="Try it out">
        Try out our API using the [API Playground](https://vatis.tech/app/playground/transcribe).
    </Step>
</Steps>
