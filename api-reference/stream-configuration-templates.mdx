<p>
    Each upload creates a new stream that is based on a pre-defined stream configuration template.
</p>
<p>
    The configuration templates dictates how a stream is structured and how the data is processed.
</p>
<p>
    You can find below the pre-defined stream configuration templates that are available for use.
</p>

<AccordionGroup>
    <Accordion title="File-optimized transcribe">
        Stream Configuration Template Id: `668115d123bca7e3509723d4`<br/>

        Input data:
          - audio data: `wav` `mp3` `mp4` `m4a` `webm` etc.
          - public `http` or `https` link to audio file

        Output data: `final transcription packets`<br/>

        | Query parameter | Default value | Description |
        | --- | --- | --- |
        | `language` | None | The language as country [ISO code](/api-reference/auxiliaries/transcription-languages) to transcribe the audio to. When not specified, it is automatically detected |
        | `noSpeechThreshold` | 0.5 | The threshold value to consider a segment as a speech segment. The value ranges from 0 to 1 |
        | `vad` | true | Enable or disable voice activity detection. Possible values: `true` or `false` |
        | `splitStereo` | false | Enable or disable splitting stereo audio into two mono audio streams. Possible values: `true` or `false` |

    </Accordion>
    <Accordion title="File-optimized transcribe with diarization">
        Stream Configuration Template Id: `669fee048452492b62cfad39`<br/>

        Input data:
          - audio data: `wav` `mp3` `mp4` `m4a` `webm` etc.
          - public `http` or `https` link to audio file

        Output data: `final transcription and diarization packets`<br/>

        | Query parameter | Default value | Description |
        | --- | --- | --- |
        | `language` | None | The language as country [ISO code](/api-reference/auxiliaries/transcription-languages) to transcribe the audio to. When not specified, it is automatically detected |
        | `noSpeechThreshold` | 0.5 | The threshold value to consider a segment as a speech segment. The value ranges from 0 to 1 |
        | `vad` | true | Enable or disable voice activity detection. Possible values: `true` or `false` |
        | `speakersNumber` | None | The number of speakers to diarize the audio to. When not specified, it is automatically detected |
        | `minSpeakersNumber` | None | The minimum number of speakers to diarize the audio to |
        | `maxSpeakersNumber` | None | The maximum number of speakers to diarize the audio to |

    </Accordion>
